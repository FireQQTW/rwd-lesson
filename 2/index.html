HTML 5

什麼是 HTML 5？

1. 由 W3C 完成標準制定。

2. 取代 HTML 4.01 和 XHTML 1.0

3. 廣義包刮 CSS 及 Javascript

4. 新增很多語法(video、canvas)

5. 刪除不必要的標籤(font、center)

支援程度

HTML5 仍然在持續的不斷修正中。

新增標籤(內容)

  • <header>: 區塊的標題或者摘要。
  • <footer>: 置底版權區域。
  • <nav>: 導覽列。
  • <section>: 內容區塊。
  • <article>: 跟section類似，但更具獨立性。
  • <aside>: 附加資訊。
  • <hgroup>: section內的標題。
  • <figure>: 內容區塊的一種，但拿掉不影響整體內容的表達
  • <figcaption>: figure元素的標題。

你也很常誤用嗎?

影音功能

  • <video>: 影片(範例)
  • <audio>: 聲音(範例)

動畫、圖像

  • <canvas>: 動畫、圖型(範例)
  • <svg>: 圖像(範例)
  • webGL: 透過JS，渲染互動式3D、2D圖形的技術。
